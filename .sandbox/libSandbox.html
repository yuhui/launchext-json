Reactor Extension Library Sandbox

How to test:

  1. Open the browser console.
  2. To test the "Parse" data element: run _satellite.getVar('parsed sampleString'), should get an object.
  3. To test the "Stringify" data element: run _satellite.getVar('stringified sampleObject'), should get a string.
  4. To test the "null" data element: run _satellite.getVar('parsed nullString'), should get null object.
  5. To test the undefined data element: run _satellite.getVar('stringified undefinedObject'), should get undefined object.
  6. To test the "undefined" data element: run _satellite.getVar('parsed undefinedString'), should get undefined object.
